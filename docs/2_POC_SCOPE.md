# POC Scope (Hackathon MVP)

To win the hackathon, we need a polished, functional demonstration of the core loop.

## 1. User Roles (Unified Mobile App - Expo)
-   **NGO/Donor Dashboard:** Post availability (e.g., "100 Blankets in Shimla"). List current "Outstanding Needs" matching their category.
-   **Field Worker/Needy Portal:** Simple mobile form for "Needs". Includes **QR code generation** for receiving aid.
-   **Doctor Dashboard:** Real-time **Video/Chat helpline** for emergency consultations.
-   **Blood Donor Dashboard:** Quick toggles for specific blood type matching.

## 2. Core Features (The "Himachal" Bundle)
-   **AI Matchmaker:** Using Ollama for context-aware resource scoring.
-   **Mobile RAG Assistant:** Small LLM helper for offline disaster guidance in Hindi/English.
-   **QR Supply Tracking:** Track from "NGO Scanning Pickup" to "Needy Scanning Receipt".
-   **Medical Triage:** Live Doctor chat and Blood Donor local alerts.

## 3. The "Wow" Factor:
-   **Multilingual First UI:** Seamless Hindi-English toggles for all help forms.
-   **Offline-Ready AI:** Ollama running locally ensures the "AI Expert" works without internet.

## Out of Scope (For Future)
-   Drone logistics integration.
-   Full blockchain-based audit trail.
-   Multi-hop supply chain tracking.
